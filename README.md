# Akash Agarwal - Brutalist Retro Terminal Portfolio

A single-page React portfolio rendered as a rigid **AKASH_OS/98** terminal desktop. The interface boots through a simulated BIOS sequence before loading an interactive OS dashboard with a command terminal, desktop executables, a live IST clock, system stats, and a database-style projects table.

## Features

- React state-driven two-phase experience: boot sequence first, main OS after 2.5 seconds.
- Interactive terminal with local history and supported commands: `help`, `whoami`, `skills`, `projects`, and `clear`.
- Tailwind CSS brutalist styling: pure black panels, phosphor green text, harsh 1px borders, monospace typography, and no rounded corners.
- Live Indian Standard Time clock in the top system bar.
- Desktop-style executable link to the classic portfolio.
- Responsive grid dashboard with sidebars, terminal window, and projects database table.

## Technologies Used

- React
- Vite
- Tailwind CSS

## Usage

Install dependencies and run the local development server:

```bash
npm install
npm run dev
```

Create a production build:

```bash
npm run build
```
